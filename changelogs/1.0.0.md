# Version 1.0.0

## Initial NeoForge Conversion and Fixes

This version marks the conversion of Time Vortex Mod from Forge to NeoForge 1.21.1, along with critical bug fixes.

### Major Changes
- **Port to NeoForge**: Complete migration from Minecraft Forge to NeoForge
- **Minecraft Version**: Updated to support Minecraft 1.21.1
- **Rematerialization Fix**: Resolved the longstanding "stuck on rematerializing" bug by implementing correct chunk forcing during the animation sequence.
- **Texture Fixes**: Restored missing textures for TARDIS Key, Wrench, and Size Designator by updating model loaders to the new NeoForge standard.

### Technical Notes
- **Event Bus Fixes**: Corrected event bus registration for chunk tickets to resolve server startup crashes.
- **Packet Handling**: Fixed decoding issues in `ClientboundMonitorDataPacket` to improved client-server communication stability.
